Due moduli da 8 ore (acquistabili separatamente) che costituiscono un percorso di formazione completo sul Cloud Computing e AWS.

Modulo 1: Corso di Cloud Computing

8 ore - giovedì 2 maggio 2013

L'obiettivo del corso è la comprensione delle metodologie principali per la progettazione di applicativi scalabili, partendo con una analisi dei prodotti Amazon Web Services, proseguendo attraverso laboratori pratici dedicati all'utilizzo di istanze virtuali, deploy di web app sul cloud e scalabilità dei database.

A chi è rivolto: sviluppatori, solution architect e responsabili IT che desiderano iniziare a lavorare con il Cloud Computing ed acquisire un'ottima conoscenza del Cloud Computing

Competenze richieste: fondamenti di programmazione e conoscenza di un linguaggio di programmazione o scripting (quale PHP, Javascript o JAVA)

Cosa comprende: corso di 1 giornata, coffe break, pranzo, materiale didattico, connettività Wi-Fi, gadget e attestato di partecipazione
